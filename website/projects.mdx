# EF JS Projects

The Ethereum Foundation JavaScript team maintains the following projects:

## Web3.js

Ethereum JavaScript API

- [GitHub](https://github.com/ethereum/web3.js)

## Ethers.js

Complete Ethereum wallet implementation and library in JavaScript

- [GitHub](https://github.com/ethers-io/ethers.js/)

## EthereumJS

The EthereumJS community builds Javascript tools implementing core Ethereum technologies, protocols and APIs for helping developers to interact with the Ethereum network and build their own applications.

- [GitHub](https://github.com/ethereumjs/)

## Ethereum Grid

Download, configure, and run Ethereum nodes and tools

- [GitHub](https://github.com/ethereum/grid)
