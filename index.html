I'm here to learn the basics.

Ethereum's homepage, ethereum.org, has a lot of great resources for getting off the ground.

This team also writes an Ethereum 101 blog series that you may find useful.

I'm trying to build a dApp.

Traditional web applications are one JavaScript module away from communicating with the Ethereum blockchain. This team maintains two such packages, Web3.js and ethers.js. Choosing one over the other is a matter of personal preference, so check out each and decide which resonates best with you. You'll find tutorials leveraging each library at ethereum.org/developers.

I'd like to contribute.

For now, each project has its own contribution tips:

  • EthereumJS
  • Web3.js
  • Ethers.js
  • Ethereum Grid

In the future, we hope to standardize on more practices, making it easier to contribute to any of these tools.

How did I get here?

We're not sure, but we're sure glad you came. If you're looking for context, this page is maintained by the JavaScript team at the Ethereum Foundation. We contribute to a wide range of EF efforts, but are the primary maintainers of Web3.js, ethers.js, EthereumJS, and Ethereum Grid. You can learn more about these projects here.
